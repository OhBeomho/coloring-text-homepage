# style

The function that makes your text styled.

`style(text, ...styles)`

## Parameters

| Parameter | Description                  |
| --------- | ---------------------------- |
| text      | The text that will be styled |
| styles    | Styles                       |

<details>
  <summary><strong>Available styles</strong></summary>

- bright
- dim
- underscore
- blink
- reverse
- hidden
</details>

## Returns

Styled text.
